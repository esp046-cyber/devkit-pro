Color Psychology for Marketing

Blue

Trust, authority, professionalism. Ideal for B2B, SaaS, finance, coaching.

Best CTAs: Learn More, Book Call, Get Started

Energy, enthusiasm, urgency. Great for fitness, e-commerce, consumer apps.

Best CTAs: Buy Now, Claim Offer, Join Free

Growth, wealth, health. Perfect for finance, health/wellness, sustainability.

Best CTAs: Save Now, Start Free, Grow Revenue

Creativity, luxury, wisdom. Great for coaches, creatives, premium brands.

Best CTAs: Apply Now, Get Access, Transform
Red

Conversion Layout Rules

✓ DO
•Place primary CTA above the fold on every key section
•Use whitespace aggressively — let content breathe
•One primary action per section, secondary in ghost/outline
•Break long copy with icons, numbers, or visuals
•Use social proof near CTAs (counters, photos, logos)
•Sticky nav with CTA on scroll for longer pages
•60px+ touch targets on mobile
✗ DON'T
•Use sliders/carousels as hero — they kill conversions
•Have multiple competing CTAs at the same hierarchy
•Use stock photos of fake smiling people
•Put auto-play video or audio
•Use light gray text on white — WCAG contrast failure
•Add navigation links that lead away from the funnel
•Use more than 3 font families on one page

Mobile-First Best Practices

📱
Thumb-First Design
Place key CTAs in the bottom 60% of screen where thumbs reach
⚡
Speed < 3s
53% of users leave if a mobile page takes more than 3 seconds
🔤
Min 16px Body
Below 16px iOS auto-zooms inputs. Use text-base as minimum
🖼️
Lazy Load Images
Use loading="lazy" and next-gen formats (WebP/AVIF)

SEO Checklist

Core Web Vitals & Speed Tips

LCP < 2.5s
Largest Contentful Paint
• Preload hero image with <link rel="preload">
• Use srcset for responsive images
• Avoid hero background-image CSS (use <img> instead)
• Self-host critical fonts
INP < 200ms
Interaction to Next Paint
• Defer non-critical JS with defer/async
• Avoid heavy event listeners on scroll
• Use passive: true on scroll events
• Break long tasks (>50ms) into smaller chunks
CLS < 0.1
Cumulative Layout Shift
• Always set width & height on images
• Reserve space for ads/embeds
• Avoid inserting content above existing DOM
• Use font-display: swap for web fonts
